# Pokretanje 🚀

- `docker-compose up -d`
- `npm i`
- `dotnet run` za pristup preko browsera @ `localhost:5001`

  ili
- `electronize start` za pokretanje desktop aplikacije

# Izgradnja 🏗
- `electronize build /target win`
- `electronize build /target osx`
- `electronize build /target linux`
